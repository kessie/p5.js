Introduction to p5.js

By Kessiena (http://p5js.org/get-started)

p5.js is a JavaScript library that starts with the original goal of Processing - to makecoding accessible for artists, designers, educators and beginners. It is designed for artists, developers and media connoisseurs to create interactive art, games, data visualizations and animations. Because p5.js is written in Javascript, it works very well on the web and can interact well with existing web-based technologies such as sound, video, webcam input, etc.

For novices in programming, learning p5.js is a great place to start. It is designed to make coding accessible for beginners and includes a supportive online community. Experienced programmers on the other hand, can achieve great things with p5.js.

Computer programming is the new literacy of the digital age and it is important that anyone who uses a computer should learn how to "talk" to a computer. According to Doug Rushkoff's book, "Program or Be Programmed" he talks about how the society is becoming increasingly dependent on machines, yet decreasingly capable of making or even using them effectively. Programming is a very important skill beside its usefulness in computer interaction, it teaches you how to think "Everyone should learn how to program a computer because it teaches you how to think." (Steve Jobs).

In P5.js, there is need to download a code editor, which is where all code is written. P5.js is currently building a code editor designed to work specifically with P5.js, In this case however, we shall be doing all coding and running on the interactive website to make it easier for novice programmers.

Using jsFiddle

JSFiddle is an online Integrated DEvelopment Environment(IDE), that is hosted in a web browser allowing users to create and execute code written in javascript with the option of using one of the java frameworks or no frameworks, html ad css. JSFiddle also shows how javascript, css and html work together but in this case we are only concerned with writing and executing code in javascript.

As can be seen in the example below the jsfiddle example has been embedded in the webpage. In other to run that example the following steps are taken:

  • Click on the "edit in jsfiddle" button at the top right corner of the embedded page, this opens the jsfiddle editor on a new webpage
  • After the fiddle editor has been opened, click on the run button to run code in javascript section.

As can be seen in the jsfiddle webpage there are four different sections to write code, html, css, javascript and the result. As explained earlier on there is only need to write and edit code in the javascript section as p5.js makes use of the javascript library.

Writing our First Simple Program

There are two main functions used when writing a program in P5.js. The setup() block runs only once and is normally used for initialization(setting up a starting value of a variable) or for greating a program that needs no loop. The draw() block runs repeatedly and is used for animation.

A function can be defined as a named section of a program that performs a specific task. In this sense, a function is basically a type of procedure or routine which performs some operation but returns no value. Although most programs have a prewritten set of functions in a library, it is possible for new functions to be written to perform specific tasks.

An example of some prewritten functions in p5.js are;

  • arc()
  • ellipse()
  • line()
